# 小丸工具箱粉丝致敬版

## 前言
先感谢一下这个工具箱原作者，因为我用小丸工具箱用了有近10年，所以，在其精神之下，出于兴趣重写这个软件，尽量实现原版功能和选项，同时扩展更新。为什么用这个名字，因为功能都是基本复刻原版的，也是对原作者的尊敬，不能致敬了还说是独立原创无参考吧，想法比代码更重要

## 项目简介
小丸工具箱粉丝致敬版，有Mac和Win双版本，这是是基于 Python Tkinter + FFmpeg 的本地多功能媒体处理工具，覆盖视频压缩、音频处理、封装转换、反挤压、字幕封装、LUT 预览、MediaInfo 分析、批量任务与 Benchmark。

## 当前版本
- `v1.1.2`

## 主要功能
- 视频压缩：支持 CPU / NVIDIA / AMD / Intel(QSV) 编码器，含常用与高级模式。
- 输出变速：支持直接设置输出倍速，自动联动视频与音频速度处理。
- 导出前确认：设置中可开启导出前二次确认，开始导出或加入队列前弹窗汇总本次输出参数与滤镜/变速等关键效果。
- 编码失败兜底：可开启自动回退 `CPU H.264 / AVC (libx264)` 重试一次。
- 音频处理：支持音频转码、视频提取音频、倒放、左右声道复制。
- 常用页面：支持视频截取、分离音视频，并自动兼容带时间码/数据流的专业素材截取场景。
- 字幕页面：支持轨道保留/移除、封装、导出字幕。
- AVS 页面：支持视频/字幕/输出路径管理，支持加黑边、裁剪、截取、亮度、锐化、降噪，支持脚本预览/保存/复制，并优化中文脚本预览显示。
- 反挤压页面：支持变形素材反挤压、目标画幅与输出分辨率控制。
- 封装页面：支持无损合并、换容器与音频模式切换。
- 复古页面：支持多种老格式转码参数模板。
- LUT 调色页面：支持批量 LUT 缩略图预览。
- MediaInfo：支持关键特性指示灯（杜比、5.1、HDR、10-bit、4K、高帧率、多音轨），支持 A/B 参数对比弹窗、CSV 导出、静态帧导出。
- 隐藏水印：视频页支持嵌入隐藏水印（文字/图片）并解析本软件写入的隐藏签名。
- 任务管理：支持入队、并发、拖动排序、停止与日志查看。
- Benchmark：支持环境检测、编码器基准测试与快捷勾选（N 卡常用/A 卡常用/I 卡常用/全选/清除）。
- 预设系统：支持“内置预设 + 用户预设”，并可在设置页管理用户预设（导入/导出/删除/排序）。
- 界面体验：支持日间/黑暗模式切换、首次使用向导（高亮引导关键按钮）、界面文字大小切换，以及弹窗相对主窗口居中显示。

## 配置与数据目录
程序运行后会在以下目录保存配置和用户数据：

```text
%APPDATA%\marurebuild
```

主要包含：
- `app_settings.json`（全局设置）
- 用户预设文件（及备份）
- 运行相关配置文件

## 运行要求
- Windows 10/11
- Python 3.10+
- FFmpeg / ffprobe 可用（已加入 PATH，或放在程序可访问路径）

## 启动方式
```bash
python main.py
```
或：
```bash
python 视频高性能压缩器.py
```

## 项目结构（核心文件）
- `main.py`：入口
- `视频高性能压缩器.py`：中文入口
- `gui.py`：界面与交互逻辑
- `ffmpeg_utils.py`：FFmpeg 命令构建与调用
- `config.py`：配置常量
- `data.py`：数据结构

## 说明
- 不同硬件编码器是否可用，取决于本机 FFmpeg 编译能力与驱动支持。
- 若硬编失败，可切换 `CPU H.264 / AVC (libx264)` 进行兜底。
